# Login System Using HTML

##  Project Description
This project is a simple Login System created using only HTML.  
It contains multiple pages connected using basic form actions and hyperlinks.

The project demonstrates basic front-end structure and navigation between pages without using CSS or JavaScript.

---

##  Pages Included

1. Login Page
2. Registration Page
3. Forgot Password Page
4. Reset Password Page
5. Dashboard Page

---

##  Features

### 1️ Login Page(index.html)
- Username field
- Password field
- Login button (Redirects to Dashboard)
- Forgot Password button (Redirects to Forgot Password page)
- Register button (Redirects to Registration page)

### 2️ Registration Page
- Name
- Email ID
- Phone Number
- Password
- Confirm Password
- Register button (Redirects to Login page)
- Login button for existing users

### 3️ Forgot Password Page
- Email input
- Send Password Reset Link button (Redirects to Reset Password page)

### 4️ Reset Password Page
- New Password
- Confirm Password
- Update Password button (Redirects to Login page)

### 5️ Dashboard Page
- Welcome message
- Logout button (Redirects to Login page)
